Для запуска программы из "py" файла сначало создадим виртуальное окружения если ипользуем vs code если используем pycharm то это делать не надо
в консоле прописываем команду "python -m venv venv" для создание виртуального окружения

потом в консоле активируем виртуальное окружения командой "venv/Scripts/active" в консоле должно появиться надпись "(venv)" зелёного цвета

затем устанавливаем нужные библиотеки из файла requirements командой "pip install -r requirements.txt"

чтобы vs code увидел созданное нами виртуальное окружения и брал библиотеки только из него при открытом "py" файле нажимаем crtl+shift+p в открывшемся откне пишем "python:Выбор интерпретатора" и выбираем который помечен как "Рекомедуемый" а не "Глобальный" теперь vs code увидет все установленые библеотеки в нашем виртуальном окружении

для создание exe файла нужно в консоле с активированным виртуальном окружением прописать команду "pyinstaller --onefile main.py",после завершения exe файл появиться в созданой папке "dist" и его можно переносить и запускать на устройствах где python не установлен
